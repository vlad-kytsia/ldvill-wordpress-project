A custom WordPress theme developed from a full Figma design for an electrical services company.  
This project includes a complete front-end implementation (HTML/CSS/JS) and full integration into WordPress using ACF, Contact Form 7, and Polylang. The site is fully responsive and optimized for mobile devices.

## ðŸ“ Project Description
This project was completed in **October 2024**.  
I converted the provided Figma design into a pixel-perfect HTML/CSS/JavaScript template and then integrated it into WordPress by creating a custom theme.

To implement dynamic content, I used **Advanced Custom Fields (ACF)**.  
The website also includes a multilingual setup using **Polylang**, a contact form created with **Contact Form 7**, and security/backup configuration using **Better WP Security** and **UpdraftPlus**.
